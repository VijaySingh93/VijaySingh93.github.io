    Projects

    Have worked on multiple projects in web development and Deep Learning. Some of these are given below.

    • Bicycle Estimation

      Built and trained a neural network to predict the number of bicycles required at a bike rental service on a given day.

    • Image Classification

      In this project, the task is to classify images from the CIFAR-10 dataset. The dataset consists of airplanes, dogs, cats, and other objects. The dataset had been preprocessed, then trained a convolutional neural network on all the samples. The images were normalized, one-hot encoded the labels, build a convolutional layer, max pool layer, and fully connected layer. At then end, predictions were made for an unknown image.

      Generated Simpsons TV scripts using RNNs. Used part of the Simpsons dataset of scripts from 27 seasons. The Neural Network generates a new TV script for a scene at Moe's Tavern.

    • Generating Faces

      Used generative adversarial networks to generate new images of faces.
